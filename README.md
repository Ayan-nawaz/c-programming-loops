# 🔁 C Programming - Loops

Welcome to my **C Programming Loops** repository! 🚀

This repository contains beginner-friendly C programs that demonstrate the use of different looping concepts. These programs are helpful for students who want to strengthen their understanding of loops and improve their programming skills.

## 📂 Programs Included

- ➕ Arithmetic Operations
- 🔄 For Loop
- 📈 Geometric Progression
- 🔢 One to Hundred
- 📋 Multiplication Tables

## 🎯 Learning Objectives

- Understand loop concepts in C
- Practice writing efficient loop-based programs
- Improve logical thinking and problem-solving skills
- Build a strong foundation for coding interviews

## 💻 Language

- C Programming

## 🛠️ Compiler

- GCC
- Visual Studio Code

## 🚀 How to Run

1. Compile the program:
   ```bash
   gcc filename.c -o output
   ```

2. Run the program:
   ```bash
   ./output
   ```
